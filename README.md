# 紫藤法道 - 法律 AI 助手

基于 Flutter 实现的法律 AI 助手《紫藤法道》前端界面。

## 功能列表

- AI 聊天界面交互
- 短视频社区
  - 瀑布流加载短视频列表
  - 短视频的上传（待实现）
  - 短视频列表搜索
  - 短视频播放界面
    - 基本的滑动页面
    - 接入视频播放 API
    - 点赞功能
    - 进度条
    - 评论区
      - 浏览与发送
- 用户功能
  - 登录与注册界面
  - 显示用户的短视频列表
  - 设置
    - 关于应用

## 应用构建指南

1. 下载 Android Studio 并按 Flutter 官方教程配置好开发环境
2. 使用 Android Studio 打开本项目
3. 在工具栏 Tools - Flutter 中运行 Flutter Pub Get
4. 编译运行应用或打包 APK
